# DecodeLabs Data Science Portfolio

Welcome to my professional Data Science and Engineering portfolio. This repository hosts an end-to-end collection of production-grade machine learning pipelines, built with rigorous statistical controls, strict data contracts, and zero-leakage validation protocols.

## Portfolio Directory

### 📁 [Project 1: Advanced EDA & Feature Engineering Pipeline](./Project_1_Data_Analysis/)
- **Core Focus:** Structural engineering of mathematical truth, rules-based missingness matrices, outlier winsorisation, collinearity eradication, and runtime data validation via `Pandera` and `Feast`.

### 📁 [Project 2: Fraud Detection for Imbalanced Financial Datasets](./Project_2_Fraud_Detection/)
- **Core Focus:** Class imbalance resolution utilizing synthetic interpolation (SMOTE) inside an isolated, leak-free `imblearn` validation pipeline with multi-algorithm benchmarking (`Logistic Regression` and `Random Forest`).

### 📁 [Project 3: Customer Segmentation via Unsupervised Learning](./Project_3_Customer_Segmentation/)
- **Core Focus:** Dimensionality reduction via Principal Component Analysis (PCA) and geometric clustering using `K-Means++` to map abstract coordinates back to actionable customer personas.
